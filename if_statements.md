處理（控制）流程 + 資料：就是程式！<br>
if 敘述是最常用的控制流程判斷式，以下是 if 敘述的內容和結構。<br>
<h3>if 敘述：</h3>
```Python
if condition expression:
	statement(s)
elif condition expression:
	statement(s)
else:
	statement(s)
```
